# Third-Party Licenses

TablesPro only allows dependencies with OSI-approved licenses suitable for free self-hosting and commercial use.

Allowed licenses:

- MIT
- Apache-2.0
- BSD-2-Clause
- BSD-3-Clause
- ISC
- 0BSD
- PostgreSQL
- CC0-1.0
- Unlicense

Run `npm run license:check` after installing dependencies to validate the installed dependency tree.
